# 中文出版校对参考标准

## 标点规范

| 类别 | 出版标准 | 常见错误 |
|------|----------|----------|
| 引号 | `""` 双层引号，内嵌用 `''`（大陆标准） | 混用、嵌套错误。非台港出版物不得使用「」『』 |
| 书名号 | 《》（篇名用〈〉） | 不用书名号或用错层级 |
| 省略号 | ……（六个点，占两格） | ... 或 …（三个点） |
| 破折号 | ——（两个长横占两格） | - 或 —（单横） |
| 连接号 | –（en dash）或 —（em dash）视语境 | 与连字符 - 混用 |
| 间隔号 | ·（居中圆点） | 与 . 或 · 混用 |
| 分号 | ； | 与英文 ; 混用 |

## 数字用法

- 出版标准一般用汉字数字（非科技类）
- 年代、百分比用阿拉伯数字
- 成语、惯用语用汉字

## 翻译腔常见模式

| 模式 | 例句（原文） | 建议修改 |
|------|-------------|----------|
| 被……所…… | "他被命运所抛弃" | "他遭命运抛弃" |
| 是……的 | "这本书是值得一读的" | "这本书值得一读" |
| 一个……的 | "一个黑暗的、潮湿的夜晚" | "一个潮湿黑暗的夜晚" |
| ……着……着 | "他走着走着，突然停了下来" | "他走了一阵，突然停下" |
| 开始……起来 | "他开始跑起来" | "他拔腿就跑" |
| 过于正式的代词 | "该计划""该事件" | "这个计划""这件事" |
| 被动语态过滥 | "他的故事被人们传颂着" | "人们传颂着他的故事" |

## 专有名词统一

- 同一个人名在同一译本中必须完全一致
- 音译名优先参考商务印书馆《英语姓名译名手册》
- 地名参考中国地图出版社《世界地名译名手册》
- 系列作品专有名词以第一部已出版译本为准
