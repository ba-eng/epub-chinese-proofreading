# Known Limits

1. Primitive accumulation remains excluded because Chinese-side stabilization is insufficient.
2. class-being / human-essence remains blocked.
3. machines fragment remains guarded-only and outside the package core.
4. some working-only term pressure remains inside approved modules and must not be mistaken for final canonical terminology.
5. this package is not a final authoritative Marx publication and must not be presented as canonical closure.
6. this package does not claim complete coverage of the Marx corpus, all period layers, or all unresolved textual and terminological disputes.
